## Repository layout

| Path | Role |
|------|------|
| **`sources/*.md`** | Source of truth for blog posts. Edit here; do not hand-edit generated HTML. |
| **`blog/*.html`** | Generated post pages. Regenerated by `ruby scripts/build_posts.rb`. |
| **`index.html`** | Homepage (build script updates the Thoughts list between HTML comments). |
| **`site.css`** / **`site.min.css`** | Styles: edit `site.css`, then run `ruby scripts/minify_css.rb` to refresh the minified file. |
| **`js/`** | Theme scripts (`theme-init.js`, `theme-toggle.js`). |
| **`assets/`** | Static files served by the site: `favicon.svg`, `og-image.png`, and **`assets/fonts/`** (Inter variable font used as a fallback by `scripts/generate_og_image.py` when system fonts are unavailable—not loaded by the live site). |
| **`scripts/`** | Build and tooling: `build_posts.rb`, `minify_css.rb`, `generate_og_image.py`. |

Performance / CSS workflow: see [PERFORMANCE.md](PERFORMANCE.md).
